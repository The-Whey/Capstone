# the store 

- npm config set @here:registry https://repo.platform.here.com/artifactory/api/npm/maps-api-for-javascript/
- npm install
- modify connection string
- create database
- npm run start:dev
- npm run start:dev:sync (builds tables and syncs data)
- npm run build:dev
